h1. ForgeosCMS

ForgeosCMS is an extension of Ruby on Rails for CMS development

VERSION : 0.9.0-rc2

h1. Prerequisites

h2. Plugins

* ForgeosCore

h2. Initialize

  <pre><code>
  $ ./script/generate migration install_forgeos_cms_plugin
  </code></pre>

Edit the generated file like this :

  <pre><code>
  class InstallForgeosCms < ActiveRecord::Migration
    def self.up
      migrate_plugin :forgeos_cms, LAST_MIGRATION
    end

    def self.down
      migrate_plugin :forgeos_cms, 0
    end
  end
  </code></pre>

Note: LAST_MIGRATION is the number of the last database migration of ForgeosCMS.

Launch FogeosCMS Install :

  <pre><code>
  $ rake forgeos:cms:install
  </code></pre>

h2. Update routes.rb

Replace your routes by :
  <pre><code>
  map.routes_from_plugin(:forgeos_cms)
  </code></pre>
