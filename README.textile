h1. ForgeosCMS

ForgeosCMS is an extension of Ruby on Rails for CMS development

VERSION : 1.9.1

h1. Prerequisites

h2. Gem

* forgeos_core >= 1.9.2

h2. Initialize
  <pre><code>
  $ rails new -d mysql my_site
  $ cd my_site
  </code></pre>
  add following gem dependencies to Gemfile
  <pre><code class="ruby">
gem 'rmagick'
gem 'acts-as-taggable-on', '>= 2.0.6'
gem 'acts_as_commentable', '>= 3.0.1'
gem 'acts_as_list', '>= 0.1.3'
gem 'acts_as_tree', '>= 0.1.1'
gem 'bcrypt-ruby', '>= 2.1.4'
gem 'fastercsv', '>= 1.5.4'
gem 'globalize3', '>= 0.1.0'
gem 'haml', '>= 3.1.2'
gem 'sass', '>= 3.1.4'
gem 'mime-types', '>= 1.16'
gem 'thinking-sphinx', '>= 2.0.5'
gem 'webpulser-habtm_list', '>= 0.1.2'
gem 'will_paginate', '~> 3.0.pre2'
gem 'authlogic', '>= 3.0.3'
gem 'forgeos_core', '>= 1.9.2'
gem 'forgeos_cms', '>= 1.9.1'
  </code></pre>
  <pre><code>
  $ rake forgeos:cms:install
  </code></pre>
